# **App Name**: Cooking Ina

## Core Features:

- Smart Ingredient Tracker: An interactive UI tool that allows users to input ingredients as dynamic chips with real-time autocomplete suggestions.
- AI Recipe Engine: An AI-powered reasoning tool that suggests recipes based on available ingredients, providing match percentages and missing items.
- Personal Recipe Vault: Secure storage of favorite dishes and search history using Firestore for a personalized culinary journey.
- Chef's Portal Auth: Secure onboarding using Firebase Authentication with Google Sign-In and standard credentials.
- Recipe Insight Details: Dynamic modals showing cooking instructions, estimated time, and integrated YouTube tutorial links for each recipe.
- Curated Discovery Filters: Tailored categorization logic to filter recipes by Breakfast, Healthy, or regional cuisines like Filipino dishes.
- Interactive Dashboard: A responsive, glassmorphic overview page featuring recent searches and trending ingredient combinations.

## Style Guidelines:

- Primary Color: Forest Mint (#14856E), reflecting culinary freshness and precision. Darker green tones chosen to maintain high contrast in a light scheme.
- Background Color: Crisp Alpine (#F5F8F8), a heavily desaturated variant of the primary hue for a clean, minimalist feel.
- Accent Color: Spring Leaf (#2DA446), an analogous hue that provides vibrancy for action buttons and high-match percentage labels.
- Font: 'Poppins', a geometric sans-serif that delivers a contemporary and fashionable aesthetic, perfect for the clean lines of recipe cards and dashboards.
- Outline-style iconography from the Phosphor collection to maintain a high-end agency look that isn't too literal.
- Modern glassmorphism cards with rounded corners (16px radius) and subtle elevation using tiered shadows to create a layered workspace feel.
- Sophisticated micro-interactions using Framer Motion for list reordering of ingredients and layout transitions between dashboard states.
